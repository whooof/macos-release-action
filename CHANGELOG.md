### Added
- Initial release
- Version bump action with semver support (patch/minor/major/custom)
- Cargo.toml update action for Rust projects
- Info.plist update action
- Xcode project update action for Swift projects
- App bundle creation with icon support
- Universal binary support (arm64 + x86_64)
- Code signing (ad-hoc and Developer ID)
- Apple notarization support (optional)
- Git tagging action
- GitHub Release creation action
- All-in-one main action
